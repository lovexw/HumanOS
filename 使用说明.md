## 💻 如何使用

### 基础浏览

1. **打开网站**：在浏览器中访问网站地址
2. **浏览器官**：滚动查看所有器官卡片
3. **查看详情**：点击感兴趣的器官
4. **阅读信息**：在弹窗中阅读详细内容
5. **关闭弹窗**：点击右上角 × 或弹窗外区域

### 筛选功能

1. 点击顶部的分类按钮：
   - 全部器官
   - 重要器官
   - 消化系统
   - 呼吸系统
   - 感觉器官
   - 内分泌系统

2. 器官列表会自动筛选显示对应类别

### 搜索技巧

虽然没有搜索框，但你可以：
- 使用浏览器搜索功能（Ctrl+F 或 Cmd+F）
- 直接在页面中搜索器官名称

## 🔄 更新维护

### 内容更新

如需更新器官信息：

1. 编辑 `script.js` 文件
2. 找到 `organData` 对象
3. 修改对应器官的内容
4. 保存并重新部署

### 样式更新

如需修改外观：

1. 编辑 `styles.css` 文件
2. 修改 CSS 变量或具体样式
3. 保存并重新部署
