## 📖 项目简介

这是一个专业的健康科普网站，全面介绍人体20多个主要器官的功能、保养方法和注意事项。网站采用现代化设计，优雅美观，完全响应式，适合在任何设备上浏览。

**网站特色：**
- 🔬 **科学专业** - 基于医学知识和公开健康资料
- 🎨 **设计优雅** - 现代渐变色，流畅动画，视觉愉悦
- 📱 **完美适配** - 手机、平板、电脑全设备支持
- ⚡ **极速加载** - 纯静态网站，全球CDN加速
- 🆓 **完全免费** - 部署在Cloudflare Pages，无任何费用

## 🚀 快速开始

### 在线访问
部署后，你的网站将可以通过 `https://你的项目名.pages.dev` 访问

### 本地预览
```bash
# 克隆或下载项目
cd health-guide

# 使用Python启动本地服务器
python -m http.server 8000

# 访问 http://localhost:8000
```

### 部署到Cloudflare Pages

**方法一：直接上传（最简单）**
1. 访问 https://dash.cloudflare.com/
2. 进入 Pages > Create a project > Upload assets
3. 上传项目文件夹
4. 完成！立即获得一个在线网站

**方法二：Git集成（推荐）**
1. 将代码推送到GitHub/GitLab
2. 在Cloudflare Pages连接你的仓库
3. 点击 "Save and Deploy"
4. 每次Git推送自动部署

详细步骤请查看 [部署指南](DEPLOYMENT.md) 或 [快速开始](QUICKSTART.md)

## 🎨 自定义

### 修改颜色主题

编辑 `styles.css` 文件开头的CSS变量：

```css
:root {
    --primary-color: #3b82f6;      /* 主色调 */
    --secondary-color: #10b981;     /* 辅助色 */
    --accent-color: #f59e0b;        /* 强调色 */
}
```

### 添加新器官

1. 在 `script.js` 的 `organData` 对象中添加数据
2. 在 `index.html` 的器官网格中添加卡片
3. 保存并重新部署

### 修改内容

所有器官的详细信息都在 `script.js` 文件的 `organData` 对象中，直接编辑即可。
